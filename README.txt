地質調査技士 現場調査部門 学習アプリ

使い方:
1. ZIPを解凍します。
2. フォルダ内の index.html を開きます。
3. 画面上部の年度切替から、令和7年度・令和6年度・令和5年度・令和4年度・令和3年度・令和1年度・平成30年度・平成29年度・平成28年度を選びます。
4. 各問で(1)〜(4)を選び、「採点する」を押します。
5. 図表・イラスト問題は各問の「PDF紙面を表示」を開いて確認してください。

注意:
- index.html、r06.html、assets フォルダは同じ場所に置いてください。
- iPhoneでうまく開けない場合は、PCで解凍してブラウザで開くのが確実です。
